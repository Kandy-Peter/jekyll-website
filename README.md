# jekyll-website
Create simple blog website using jekyll in ruby.
Jetkyll is a ruby gme for blog post and statcic website generator

## Prerequise
```
 - Ruby(ruby -v)
 - RubyGems(gem-v)
 - GCC and Make(gcc-v. make -v, g++ -v)
```

### Install

```
  gem install jekyll bundler
```

#### commands

```
  bundle install
  bundle exec jekyll serve
```
